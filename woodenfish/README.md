## 使用

### 本地安装

将源码拉到本地：

```
git clone https://github.com/poboll/fish.git
```

或下载`.zip`压缩包，并解压到本地：

```
https://github.com/poboll/fish/archive/refs/heads/main.zip
```

打开`index.html`文件，开始积攒功德吧！
